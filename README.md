## 🚀 Getting Started

### 📋 Prerequisites

- **Python 3.6+** (Recommended: Python 3.8+)
- **Git** for version control
- **Virtual environment** (venv, conda, or pipenv)
- **8GB+ RAM** (recommended for deep learning tasks)

### ⚡ Quick Setup

```bash
# 1️⃣ Clone the repository
git clone https://github.com/M-Sarim/codealpha_tasks.git
cd codealpha_tasks

# 2️⃣ Create virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# 3️⃣ Install dependencies for specific project
cd Task1_Titanic_Classification
pip install -r requirements.txt

# 4️⃣ Launch Jupyter Notebook
jupyter notebook
```

### 🐳 Docker Setup (Optional)

```bash
# Build Docker image
docker build -t codealpha-datascience .

# Run container
docker run -p 8888:8888 -v $(pwd):/workspace codealpha-datascience
```

## ⚡ Quick Run Guide

### 🎯 **Run All Projects Sequentially**

```bash
#!/bin/bash
# Automated project execution script

projects=("Task1_Titanic_Classification" "Task2_Data_Preprocessing"
          "Task3_Leaf_Disease_Detection" "Task4_A_B_Testing_Analysis")

for project in "${projects[@]}"; do
    echo "🚀 Running $project..."
    cd "$project"

    # Install dependencies
    pip install -r requirements.txt

    # Run main script (if available)
    if [ -f "main.py" ]; then
        python main.py
    fi

    # Return to root directory
    cd ..
    echo "✅ $project completed!"
done

echo "🎉 All projects executed successfully!"
```
